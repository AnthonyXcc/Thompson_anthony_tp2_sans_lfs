# Thompson_anthony_tp2

Developed with Unreal Engine 5

platformer avec des abilités de mouvement comme requiered pour progresser.

abilité comme double saut, sprint, graple

premier lvl gagne sprint pour avoir une cléf, utilise la clef pour entrer dans une tour. embiance de foret le jour.

deuxieme lvl grimper la tour avec un graple(pas certain) et un double jump pour ouvrir un coffre et finir le jeu ambience sombre, tour en roche.
