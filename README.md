# Thompson_anthony_tp2

Developed with Unreal Engine 5

platformer avec des abilités de mouvement comme requierement pour progresser.

abilité comme double saut, sprint, graple

premier lvl gagne sprint pour avoir une cléf, utilise la clef pour entrer dans une tour

deuxieme lvl grimper la tour avec un graple(?) et un double jump pour ouvrir un coffre et finir le jeu
